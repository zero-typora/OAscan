# OAscan

![image-20240126185722494](/Users/admin/any/%E5%AD%A6%E4%B9%A0/golang/image-20240126185722494-6266644.png)

用友漏洞批量检测，poc检测难免有误报的可能在。

# 单个检测
```go
OAscan.exe -u url -o 1
```

# 批量检测
```go
OAscan -f target.txt -o 1
```
